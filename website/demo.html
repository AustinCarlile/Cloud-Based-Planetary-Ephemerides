Diagram of the Proposed Solution:


As you can see in the diagram above, the user will interact with a web service that returns ISD images

The process is as follows:

  1. The user queries the web service on the Amazon AWS server for an ISD (Image Support Data) image
  2. The web service checks if the ISD image is on the caching server, a parallel Amazon AWS server
  3. If the ISD image is on the caching server, it is returned to the user
  4. If the ISD image is not on the caching server, the logic flows as follows:
  5. The web service uses the ALE (Abstraction Layer for Ephemerides) library to request ISD images from the NASA SPICE Database
  6. The ALE library returns the ISD image to the web service
  7. The web service then sends the ISD image to the caching server and to the user
  8. The caching server will periodically use the ALE library to check for any SPICE data changes and update the cache
